Quick Start

Follow these steps to bring up a local kcore environment and launch your first virtual machine.

1) Install Binaries

curl -sSL https://get.kcore.systems | sh
kctl version

2) Start the Controller

cat < controller.yaml
listen_address: "0.0.0.0:50051"
database: "./kcore.db"
EOF

kcore-controller --config controller.yaml

3) Configure the CLI

mkdir -p ~/.kcore
cat < ~/.kcore/config.yaml
controller_address: "localhost:50051"
insecure: true
EOF

4) Create a VM

kctl create vm web-server \
  --memory 2048 \
  --vcpu 2 \
  --disk size=20GB

kctl get vms
