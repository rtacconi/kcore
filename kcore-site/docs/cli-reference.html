CLI Reference

Common `kctl` commands for day-to-day cluster and VM management.

Inspect

kctl get vms
kctl describe vm my-vm
kctl get nodes

Lifecycle

kctl start vm my-vm
kctl stop vm my-vm
kctl restart vm my-vm
kctl delete vm my-vm

Create and Apply

kctl create vm app --memory 4096 --vcpu 4 --disk size=40GB
kctl apply -f vm.yaml

Configuration

cat ~/.kcore/config.yaml
kctl version
