Atomic Updates & Instant Rollback

Never worry about broken deployments again. kcore's atomic update system ensures that every change is either fully applied or completely reverted - no half-broken states.

  • ✓
    Immutable Base System
    The core OS never changes. All updates are versioned and isolated.
  • ✓
    One-Click Rollback
    Instantly revert to any previous state with zero downtime.
  • ✓
    State Verification
    Automatic health checks ensure updates are successful before committing.
Version 1 (Current)
↓
Apply Update → Version 2
↓
✓ Health Check Passed
or
✗ Health Check Failed
↓
Auto Rollback → Version 1

Declarative Infrastructure as Code

Define your entire infrastructure in code. Version control everything from VMs to networks to storage. No more clicking through UIs or running manual commands.

  • 📝
    Native Terraform Provider
    First-class integration with your existing Terraform workflows.
  • 🔄
    GitOps Ready
    Commit infrastructure changes and watch them automatically deploy.
  • 📊
    State Management
    Automatic drift detection and reconciliation keeps reality in sync. 
# Infrastructure as Code
resource "kcore_vm" "web" {
  count  = 3
  name   = "web-${count.index}"
  memory = 4096
  vcpu   = 2
  
  disk {
    size         = "50GB"
    storage_class = "ssd"
  }
  
  network {
    name = "production"
  }
}

resource "kcore_vm" "db" {
  name   = "postgres"
  memory = 8192
  vcpu   = 4
  
  disk {
    size         = "200GB"
    storage_class = "nvme"
  }
}
